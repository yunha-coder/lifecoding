4. 줄바꿈

줄을 바꾸는 태그로는 "p 태그, br 태그"가 있다.
차이는 단락를 바꿀 때는 "p 태그"를 쓰는 게 바람직하다.
그러나 p 태그는 단락간 거리가 고정되어 있다는 단점이 있다.
그래서 쓰는 만큼 줄바꿈이 되어서 간격을 조절할 수 있는 br 태그가 선호되었지만 지금은 p 태그의 단점이 사라졌다.

위와 같이 p 태그에 style="margin-top:45px"를 추가하면 p 태그의 위쪽에 45px 만큼의 여백(margin)이 생깁니다.

CSS란 p태그의 한계 극복 기능만 가지는 게 아니다. 정보를 표현하는 html와 완전 다른 문법을 가진 css는 정보를 꾸며준다.

중요한 건 p 태그로 단락의 경계를 분명히 하면서 CSS를 통해서 p 태그의 디자인을 자유롭게 변경할 수 있기 때문에 br 태그 보다 p 태그가 더 좋은 선택입니다.

5. HTML이 중요한 이유

웹은 접근성(accessibility, 누구나 차별없이 정보에 접근할 수 있어야 한다는 철학)을 가진다.
근데 이미지로 만든 제목을 붙인다거나 하면 스크린리더(screen reader)와 같은 프로그램을 이용해서 정보를 청각화해서 접하는 시각장애인은 볼 수가 없다.
그래서 html은 비즈니스적인 측면뿐만 아니라 휴머니즘적인 측면에서도 중요하다.

Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications.Web browsers receive HTML documents from a web server or from local storage and render them into multimedia web pages. HTML describes the structure of a web page semantically and originally included cues for the appearance of the document.

HTML elements are the building blocks of HTML pages. With HTML constructs, images and other objects, such as interactive forms, may be embedded into the rendered page. It provides a means to create structured documents by denoting structural semantics for text such as headings, paragraphs, lists, links, quotes and other items. HTML elements are delineated by tags, written using angle brackets.
